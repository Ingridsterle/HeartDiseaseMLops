# Heart Disease MLops Project

## Description
Ce projet vise à prédire les maladies cardiaques à partir d’un dataset contenant les features suivantes :  

`age, sex, cp, trestbps, chol, fbs, restecg, thalach, exang, oldpeak, slope, ca, thal, target`.

Le projet suit un workflow MLops complet :  
- Prétraitement des données  
- Entraînement d’un modèle de machine learning (réseau de neurones via scikit-learn)  
- Inférence / prédiction sur de nouvelles données  
- Monitoring et suivi des expériences avec MLflow  
- Versionnement des données et modèles avec DVC
